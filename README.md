# GithubUserView Based on UserName

## Using RESTAPI

#### Using Github Api : [GithubAPI](https://developer.github.com/v3/)

## Product motivation

- 다른 REST API에 친숙해지기 위해서 Description 잘 되어있는 Github-API를 선택하여 개발하기 시작 함

## Detail

- Show User Info
- Show Repo gave Star
- Show Follower / Following User
- Show My Repo
- Show Commit List
- Show Random User
